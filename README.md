# Telegram File Manager (SSH-Bot)

Файловый менеджер для удаленного управления компьютером через Telegram. Позволяет просматривать файловую структуру с помощью интерактивных ссылок прямо в интерфейсе Telegram.

## 🚀 Возможности
- **Древовидная навигация**: папки раскрываются и сворачиваются кликом по ссылкам без отправки лишних сообщений.
- **Индикатор загрузки**: визуальный спиннер при чтении больших директорий.
- **Ограничение прав**: белый список ID пользователей (`ALLOWED_USERS`).

## 🛠 Предстоящие функции (в разработке)
- `/exec <cmd>` — удаленное выполнение консольных команд
- `/get <path>` — скачивание файлов с ПК
- `/put` — загрузка файлов на ПК
- `/sys` — мониторинг CPU / RAM / жестких дисков
- `/ps` — просмотр списка активных процессов
- `/kill <pid>` — завершение процесса
- `/find <name>` — поиск файлов
- `/cat <path>` — быстрый просмотр текстовых файлов

## 📦 Установка и запуск

### 1. Требования
- Python 3.10 или выше
- Установленная библиотека `python-telegram-bot`:
  ```bash
  pip install python-telegram-bot
  ```

### 2. Настройка токена
1. Скопируйте файл конфигурации `.env.example` в новый файл `.env`:
   ```bash
   cp .env.example .env
   ```
2. Откройте `.env` и укажите ваш токен бота Telegram:
   ```env
   BOT_TOKEN=ваш_токен_бота
   ```

### 3. Запуск
Запустите бота:
```bash
python main.py
```
Откройте диалог с ботом в Telegram и отправьте `/start`.
