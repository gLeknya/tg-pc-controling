# Telegram File Manager (SSH-Bot)

Файловый менеджер для удаленного управления компьютером через Telegram. Позволяет просматривать файловую структуру с помощью интерактивных ссылок прямо в интерфейсе Telegram.

## 🚀 Возможности
- **Древовидная навигация**: папки раскрываются и сворачиваются кликом по ссылкам без отправки лишних сообщений.
- **Индикатор загрузки**: визуальный спиннер при чтении больших директорий.
- **Ограничение прав**: белый список ID пользователей (`ALLOWED_USERS`).
- **Удаленное выполнение команд**: команда `/exec <cmd>` позволяет выполнять консольные команды на управляемом ПК.
- **Передача файлов**: скачивание файлов с ПК с помощью `/get <path>` и отправка файлов на ПК (просто отправьте файл боту как документ после команды `/put`).
- **Системный мониторинг**: получение информации о CPU, RAM, дисках с помощью `/sys` и списка процессов через `/ps`.
- **Управление процессами**: завершение зависших процессов по PID с помощью `/kill <pid>`.
- **Поиск и просмотр**: поиск файлов по маске через `/find <name>` и просмотр текстовых файлов с помощью `/cat <path>`.

## 📦 Установка и запуск

### 1. Требования
- Python 3.10 или выше
- Установленные библиотеки:
  ```bash
  pip install python-telegram-bot
  ```
- *(Опционально)* Для расширенного мониторинга процессов и системы рекомендуется установить `psutil`:
  ```bash
  pip install psutil
  ```

### 2. Настройка токена
1. Скопируйте файл конфигурации `.env.example` в новый файл `.env`:
   ```bash
   cp .env.example .env
   ```
2. Откройте `.env` и укажите ваш токен бота Telegram, а также (опционально) ID разрешенных пользователей:
   ```env
   BOT_TOKEN=ваш_токен_бота
   ALLOWED_USERS=123456789,987654321
   ```

### 3. Запуск
Запустите бота:
```bash
python main.py
```
Откройте диалог с ботом в Telegram и отправьте `/start`.
