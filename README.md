# Telegram File Manager (SSH-Bot)

Удобный файловый менеджер для удаленного управления вашим компьютером через Telegram. Позволяет просматривать файловую структуру с помощью интерактивного дерева каталогов, скачивать и загружать файлы, выполнять команды и следить за состоянием системы.

---

## 🚀 Возможности и Команды

### 📂 Работа с файлами
- **Интерактивное дерево файлов**: отправьте `/start`, чтобы открыть список дисков вашего ПК. Вы можете раскрывать и сворачивать папки прямо внутри сообщения, нажимая на ссылки с символом 📁.
- **Просмотр файлов**: используйте `/cat <путь_к_файлу>` для быстрого просмотра текстовых файлов (выводит первые 4000 символов).
- **Скачивание файлов**: используйте `/get <путь_к_файлу>` для загрузки любого файла с ПК прямо в чат Telegram.
- **Загрузка файлов на ПК**: отправьте команду `/put`, чтобы получить инструкцию. Затем просто пришлите любой файл в чат как документ (файл) — бот автоматически сохранит его в последнюю открытую вами папку (или в рабочую директорию бота).
- **Поиск файлов**: используйте `/find <маска_файла> [стартовая_папка]` для быстрого поиска файлов по названию (например, `/find *.txt C:\`).

### 💻 Управление системой
- `/sys` — показывает общую информацию о системе (название ОС, загрузка процессора CPU, использование оперативной памяти RAM и свободное место на дисках).
- `/ps` — показывает список запущенных на ПК процессов (первые 100 элементов с их PID).
- `/kill <PID>` — завершает выбранный процесс по его идентификатору.
- `/exec <команда>` — выполняет любую терминальную (консольную) команду на ПК и возвращает текстовый результат её работы.

## 📦 Настройка и Запуск бота

### 1. Подготовка окружения
У вас должен быть установлен Python версии 3.10 или выше.

Установите необходимые библиотеки:
```bash
pip install python-telegram-bot
```

*(Рекомендуется)* Для работы команд мониторинга системы (`/sys`, `/ps`, `/kill`) установите библиотеку `psutil`:
```bash
pip install psutil
```

### 2. Настройка конфигурации
1. Скопируйте файл `.env.example` и переименуйте копию в `.env`.
2. Откройте `.env` в любом текстовом редакторе и укажите:
   - `BOT_TOKEN` — токен вашего Telegram-бота (можно получить у официального бота [@BotFather](https://t.me/BotFather)).
   - `ALLOWED_USERS` — (рекомендуется) укажите через запятую ID пользователей Telegram, которым разрешен доступ к боту. Если оставить пустым, ботом сможет пользоваться любой человек.
   Пример файла `.env`:
   ```env
   BOT_TOKEN=1234567890:ABCdefGhIJKlmNoPQRsTUVwxyZ
   ALLOWED_USERS=111111111,222222222
   ```

### 3. Запуск бота
Запустите бота на вашем ПК с помощью команды:
```bash
python main.py
```
Перейдите в Telegram, откройте чат с вашим ботом и отправьте `/start`.
